==========================================
  Dammy 的个人网站 — 数据中心
==========================================

这里是整个网站的内容来源。你只需要修改这里的文件，
然后告诉我「同步部署」，我就会：

1. 读取你修改的所有 .txt 文件
2. 把照片/视频/音频复制到网站对应位置
3. 自动更新网站代码
4. 部署上线

==========================================
  文件夹说明
==========================================

dammy-data/
├── profile.txt          ← 名字、标语等基本信息
├── about/
│   ├── bio.txt          ← 个人故事、价值观、身份标签
│   └── (个人头像放这里，命名为 avatar.jpg)
├── education/
│   └── timeline.txt     ← 教育经历时间线、技能
├── projects/
│   └── projects.txt     ← 项目列表、研究兴趣
├── interests/
│   └── interests.txt    ← 兴趣爱好、为你选的歌
├── photos/
│   └── (照片放这里)     ← 照片墙的图片
├── videos/
│   └── (视频放这里)     ← 本地视频文件
├── audio/
│   └── (音频放这里)     ← 播放列表的 mp3
└── contact/
    └── contact.txt      ← 邮箱、GitHub、LinkedIn

==========================================
  使用方式
==========================================

1. 修改上面任意 .txt 文件的内容
2. 把照片、视频、音频放到对应文件夹
3. 对我说：「同步部署」

我会自动处理剩下的一切。
